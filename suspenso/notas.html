Política de Privacidade

Última atualização:

1. Coleta de Dados

O sistema de notas coleta as seguintes informações:

  • Conteúdo das Notas: O texto inserido nas notas é armazenado localmente em seu dispositivo para que você possa acessar e editar suas notas posteriormente.
  • Posição das Notas: A posição da nota na tela (coordenadas x, y) é salva para garantir que a nota apareça na mesma localização na próxima vez que você acessar a página.
  • Preferências de Cor: A cor de fundo de cada nota é armazenada para personalizar a experiência do usuário.

2. Uso de Dados

As informações coletadas são usadas exclusivamente para o funcionamento da funcionalidade de notas:

  • Armazenamento local: As notas são armazenadas localmente no seu dispositivo, sem que haja necessidade de transmissão para servidores externos.
  • Ajustes de exibição: A posição das notas e as preferências de cor são usadas para manter a consistência da interface de usuário entre as sessões.

3. Não Compartilhamento de Dados

Nenhuma informação coletada através da funcionalidade de notas será compartilhada com terceiros. As notas são armazenadas localmente no seu dispositivo e não são enviadas para servidores externos, a menos que explicitamente mencionado em outra parte deste documento ou por meio de funcionalidades adicionais que você venha a ativar.

4. Armazenamento Local

As notas são armazenadas localmente no seu navegador, utilizando a tecnologia Storage API (armazenamento local) do navegador, o que significa que os dados não são enviados para servidores externos. Esses dados podem ser acessados ou apagados a qualquer momento, através das configurações do seu navegador.

5. Segurança

Embora tomemos precauções para proteger os dados armazenados localmente, não podemos garantir 100% de segurança em relação à proteção contra falhas técnicas ou ataques cibernéticos. Recomendamos que você evite armazenar informações sensíveis em suas notas.

6. Alterações nesta Política de Privacidade

Podemos atualizar esta Política de Privacidade de tempos em tempos para refletir mudanças em nossas práticas ou para atender a novas regulamentações legais. Quando ocorrerem alterações substanciais, você será notificado de maneira adequada. Recomendamos que você revise esta política periodicamente para se manter informado sobre como estamos protegendo suas informações.

7. Seus Direitos

Você tem o direito de acessar, corrigir e excluir os dados armazenados localmente. Para fazer isso, basta acessar as configurações do seu navegador e excluir os dados do armazenamento local, incluindo as notas salvas.
